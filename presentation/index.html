Introducing CSS

CSS takes care of the visual formatting

color


<p style="color: green;">This is green!</p>

<p style="color: red;">This is red!</p>

<p style="color: orange;">And this is orange!</p>

001-color.html

 

The style="..." property allows to define CSS styles on individual HTML elements.

font-size


<p style="font-size: 72px;">This is big!</p>
		
<p style="font-size: 8px; ">This is small!</p>

<p style="color: green; font-size: 72px;">And this is green and big!</p>

002-font-size.html

Defining colors


<p style="color: orange;">This is orange!</p>

<p style="color: rgb(255, 165, 0);">And this is also orange!</p>

<p style="color: #FFA500;">And again orange!</p>

001-color.html

 

Colors can be defined as

  • Named colors (list)
  • RGB colors: e.g. rgb(255, 165, 0)
  • HEX colors: e.g. #FFA500

<p style="font-family: times new roman;">Times New Roman (serif)</p>

<p style="font-family: courier new;">Courier New (monospace)</p>

003-font-family.html

What fonts can I use?

Sans-Serif fonts

Arial, Arial Black, Tahoma, Trebuchet MS, Verdana

 

Serif fonts

Georgia, Times, Times New Roman

 

Monospace fonts

Courier, Courier New

Fallback fonts


<p style="font-family: tahoma, arial, sans-serif;">Some text</p>

The browser first tries Tahoma, then Arial and finally takes a default sans-serif font.
